Model backends (beyond the UI)

You can use model APIs directly, not only editor frontends. This is often faster and easier to automate.

  • Llama: local runtimes (e.g. Ollama) for private/offline loops.
  • Claude: API + prompt files for controlled long-context sessions.
  • Gemini: API + multimodal prompts for docs and screenshots.
  • GPT: API for reliable coding assistants and test generation.

CLI + shell paths

Pick what you have: Anti-Gravity CLI, VS Code terminal, Windows CMD, or PowerShell. The method is the same: prompt → patch → run → test. 

# PowerShell
$env:MODEL_PROVIDER="gpt"
$env:MODEL_API_KEY="your_key"
python app.py
:: Windows CMD
set MODEL_PROVIDER=gemini
set MODEL_API_KEY=your_key
python app.py
