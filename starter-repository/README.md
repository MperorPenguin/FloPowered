# Starter Repository

This folder contains a ready-to-copy starter repository scaffold.

## Included
- `src/main.py`: tiny entrypoint script
- `tests/test_main.py`: basic unit test
- `.gitignore`: Python-focused ignore rules
- `LICENSE`: MIT template

## Use it as a standalone repository

```bash
cp -R starter-repository my-new-repo
cd my-new-repo
rm -rf .git
git init
git add .
git commit -m "Initial commit"
```
